¿Que requerimientos técnicos se emplean?


Videos de las Entrevistas

-Cable:Cat 5e para 1Gbps, Cat 6 para 10Gbps, Cat 6a/7 para más. La fibra óptica para distancias largas.

-Conectores:RJ-45 para Ethernet.

-Switch:El cerebro que reparte la conexión entre varios dispositivos.

-Router:Para salir a internet y gestionar la red.

-Patch Panel + Rack:Para ordenar todo si es una oficina grande.

-Herramientas:Crimpadora, probador de cable, canaletas, escalera.
